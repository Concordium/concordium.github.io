Change account name#

Concordium Wallet

  1. Go to the Your Accounts screen (tap the account identifier on the main screen).

  2. Select the account to rename and tap the gear icon in the upper right corner.

  3. On the Account settings screen, tap Change account name.

  4. Enter the new name and tap Save.

Desktop Wallet

You can edit the name of the account. Click to edit the account name. Click to save the change.

Concordium Wallet for Web

  1. Go to the Accounts page.

  2. Click the Edit button next to the account name you want to change.

  3. Edit the name and click Save.
