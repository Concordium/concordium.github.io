==========
Wallet SDK
==========

This section describes functionality provided in the Concordium  Web-SDK, Android-SDK and the Swift SDK that enables the creation of a Concordium specific wallet. This functionality is in general referred to as the Wallet SDK, but is technically just a part of the functionality exposed by these SDKs.
