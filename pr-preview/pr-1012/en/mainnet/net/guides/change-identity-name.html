Change identity name#

Concordium Wallet for Mobile

  1. Tap . Select Your identity cards.

  2. Tap the identity you want to edit.

  3. On the identity card tap next to the identity name.

  4. Change the name and tap Save.

Concordium Wallet for Web

  1. Click the hamburger menu and select ID cards. If you have more than one ID card, select the ID card you want to edit in the ID cards drop-down.

  2. Click next to the identity name.

  3. Change the name and click .
